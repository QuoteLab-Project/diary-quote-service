# ===============================
# Commit Message Template
# ===============================

# [Type] Commit Title
# 예시:
# feat: 회원가입 API 구현
# fix: JWT 토큰 검증 오류 수정
# chore: gitignore 설정 추가
# docs: README 실행 방법 추가
# build: Dockerfile 수정
# test: 회원가입 테스트 코드 추가
# refactor: 로그인 로직 리팩터링
# hotfix: 배포 서버 긴급 오류 수정

# ===============================
# Commit Title Type
# ===============================
# ✨ feat     : 새로운 기능 추가
# 🐛 fix      : 기능 수정, 버그 수정
# 💡 chore    : 오타 수정 / 설정 변경 / 주석 수정
# 📝 docs     : 문서 수정 (README 등)
# 🚚 build    : 빌드 관련 파일 수정 및 삭제
# ✅ test     : 테스트 코드 추가 (프로덕션 코드 변경 X)
# ♻️ refactor : 코드 리팩터링
# 🚑 hotfix   : 긴급 수정

# ===============================
# Commit Body (선택)
# ===============================
# - 무엇을 변경했는지
# - 왜 변경했는지
# - 영향 범위는 무엇인지
